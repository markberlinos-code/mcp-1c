# Участие в разработке

Спасибо за интерес к проекту! Мы рады любому вкладу: от исправления опечаток до новых инструментов.

## Начало работы

### Требования

- Go 1.25+
- Платформа 1С:Предприятие 8.3.x или 8.5.x (для интеграционного тестирования)

### Сборка

```bash
git clone https://github.com/feenlace/mcp-1c.git
cd mcp-1c
go build -o mcp-1c ./cmd/mcp-1c
```

### Запуск тестов

```bash
go test ./... -v -race
```

### Mock-сервер 1С

Для разработки и тестирования без реальной базы 1С используйте встроенный mock-сервер:

```bash
go run ./cmd/mock-1c -port 9191
```

Он эмулирует HTTP-сервис 1С и позволяет работать с MCP-сервером без запущенной платформы.

## Расширение 1С

Исходники расширения находятся в `extension/src/`. Скрипты сборки в `scripts/`.

Для пересборки расширения (.cfe) из исходников:

```bash
# macOS
./scripts/build-extension.sh ~/Documents/InfoBase

# Windows
scripts\build-extension.cmd C:\Users\User\Documents\InfoBase
```

## Как предложить изменения

### Баг-репорты и предложения

Используйте [Issues](https://github.com/feenlace/mcp-1c/issues) с соответствующим шаблоном:

- **Баг-репорт** - ошибки в работе сервера или расширения
- **Предложение функциональности** - новые возможности и улучшения

Перед созданием issue проверьте, нет ли уже похожего.

### Pull Requests

1. Форкните репозиторий
2. Создайте ветку от `main`: `git checkout -b my-feature`
3. Внесите изменения
4. Убедитесь, что тесты проходят: `go test ./... -v -race`
5. Откройте Pull Request с описанием изменений

## Стиль кода

- Стандартный Go-стиль, форматирование через `gofmt`
- Именование по [Effective Go](https://go.dev/doc/effective-go)
- Моки генерируются через `go generate ./...` (не пишите их вручную)
- Комментарии к экспортируемым функциям и типам обязательны

## Структура проекта

```
cmd/
  mcp-1c/        # точка входа MCP-сервера
  mock-1c/       # mock-сервер для разработки
internal/        # внутренние пакеты
server/          # реализация MCP-сервера
tools/           # реализация MCP-инструментов
onec/            # клиент к HTTP-сервису 1С
bsl/             # встроенная справка BSL
installer/       # автоустановка расширения
extension/src/   # исходники расширения 1С
scripts/         # скрипты сборки
docs/            # документация
```
